# Continue Learning

Use this playbook when the user asks to continue from the previous learning session.

Example triggers:

- "Continue where we left off."
- "Let's keep going."
- "What were we learning last time?"

## Process

1. Read `LEARNING_STATE.md`.
2. Identify the last learned topic and current learning path.
3. Check `index/topic-list.md` so the same topic is not repeated as new.
4. Check `index/prerequisites.md` for missing foundations.
5. Continue with the next useful topic in the current path.
6. If the path is blocked, teach the missing prerequisite first.

## Afterward

Update:

- `LEARNING_STATE.md`
- `index/topic-list.md`
- the relevant topic note under `topics/`
- `index/prerequisites.md` or `index/bookmarks.md` if needed
