# V.0.1

V.0.1 is a knowledge organization project for storing, editing, and indexing topics learned through chats.

The goal is to turn scattered conversations into a structured learning map. Topics are grouped into categories based on what they are about, how often they appear, and what knowledge is needed before they can be understood.

## Purpose

Chats often contain useful ideas, explanations, questions, and references, but they can become difficult to navigate over time. This project is meant to organize that information into a growing index of learned topics.

Instead of storing everything in one flat list, V.0.1 organizes knowledge by learning progression.

For example, if many topics are about science and later the focus shifts to economics, those topics should become separate categories. If a topic depends on earlier knowledge, it should be placed after the required foundations.

## Core Idea

Knowledge should be organized in the order it becomes understandable.

Just as quantum physics should not come before basic math, advanced topics should be connected to the concepts needed to understand them. When two related topics are not directly next to each other in the structure, bookmarks or links should point to the required background information.

## Organization Principles

- Create main categories when a new broad subject appears.
- Create subcategories when a subject becomes more specific.
- Preserve the order of learning where possible.
- Connect advanced topics to prerequisite topics.
- Add bookmarks when background knowledge is stored elsewhere.
- Let the structure grow naturally based on the user's needs.

## Example Categories

- Science
- Economics
- Philosophy
- Programming
- Writing

## Future Goals

- Store chat-based learning notes as editable files.
- Build an index of topics and subtopics.
- Track prerequisite knowledge for each topic.
- Add bookmarks between related ideas.
- Support progressive learning paths.
- Make it easy to find, revise, and expand previous knowledge.
